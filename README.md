# 🧠 DeskAI – Voice Controlled Desktop Assistant

DeskAI is a Python-based voice-controlled desktop assistant that allows users to perform various computer operations using simple voice commands. It is designed to improve productivity by enabling hands-free interaction with the system.

---

## 🚀 Features

* 🎙️ Voice Command Recognition
* 🔊 Control System Volume (Increase/Decrease/Mute)
* 🖥️ Open Applications & Websites
* 📊 Check Running Application Status
* 📁 File & Folder Operations:

  * Create
  * Delete
  * Move
  * Extract files
* 🎵 Play Specific Videos on YouTube
* 🔒 Lock Screen
* 📸 Take Screenshots
* ⚡ Shutdown System
* 🤖 Real-time Voice Feedback

---

## 🛠️ Tech Stack

* **Python**
* `speech_recognition`
* `pyttsx3`
* `os` / `shutil`
* `webbrowser`
* `pyautogui`
* Other system automation libraries

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Aditya120205/DeskAI.git
```

2. Navigate to the project folder:

```bash
cd DeskAI
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python main.py
```

---

## 🧠 How It Works

DeskAI listens to user voice commands, processes them using speech recognition, and executes the requested tasks using Python automation libraries. It also provides voice responses for better user interaction.
